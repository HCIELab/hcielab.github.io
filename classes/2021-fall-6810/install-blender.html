3D editing with Blender 2.8+

What is Blender?
Blender is a 3D modeling program that can be used to create 3D models.
In this class , we will use it to create models for 3D printing but it can also be used for creating 3D animations for movies or 3D game assets for digital games.
It comes with a powerful render engine that lets you generate realistic preview and even physical simulations of various materials.

Open Source Implementation + Supports Writing of Plugins
The editor is fully open source and can be used for any private or commercial purpose.
It supports writing custom plug-ins with python.
For instance, the ESP microcontroller can stream sensor data to the 3D editor and automatically modify the models. If you, e.g., streamed the data of your touch slider to Blender, you could use it to resize digital models through the physical input.

Installing Blender
Blender is available for all major operating systems.
Install the latest version of Blender from here: https://www.blender.org/download/
