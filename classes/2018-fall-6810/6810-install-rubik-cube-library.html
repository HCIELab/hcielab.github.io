6.810 Engineering Interactive Technologies (fall 2018)

HW: Install Rubik's Cube Library

Task:

For our friday skills lab please install:

1) Python 3 (Python 2.7x will not work!)
- you need the newer Python version for code to run properly, Python 2.7x will not work
- install over terminal/command line or download here: https://www.python.org/downloads/

2) Install a Python package called “kociemba”
- you can download and install from: https://github.com/muodov/kociemba.git
- or directly type “pip install kociemba” in terminal/command line
- you can check the installation by going to any python console (i.e. type “python” in terminal), and type “import kociemba”. If there is no error pop up, the installation is succeeded.
