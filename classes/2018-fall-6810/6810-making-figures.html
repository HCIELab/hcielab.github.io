Making Figures


Goal

The goal for today is to learn how to create a paper figure.
You will compose two or more images into a single image, then add (a),(b),(c) labels and other text to the figure.
While you can use our example files below, we much preferred if you made a real figure for your current paper. Just take a few screenshots or snap a photo with your phone, then use those for the tutorial below.
Once you are done, please send your example figure to Stefanie by email.



Example Files

  • please download the example files since they contain the Adobe Illustrator Template Document you need later:
  • example-figure.zip

Cropping Each Image


  • open all images you like to assemble into a figure in Photoshop



crop each image to only keep the area you like:
  • select the crop tool
  • drag the edges or corners with the little markers
  • double click the image when you are done to confirm the cropped area




Increasing the Canvas Size to Make Space for Both Images


  • in one of your images, make the canvas size larger
  • go to Image -> Canvas Size, then type in a larger number (just double or tripple the size) -> confirm



Paste your Other Images into the Increased Canvas


  • select the content of your other image: command + A or go to Menu: Select -> All
  • copy the selection: command + C or go to Menu: Edit -> Copy
  • switch over to your large canvas, then paste the content: command + V or go to Menu: Edit -> Paste

Moving / Scaling Images


  • to move an image, simply drag it on the canvas
  • to scale an image either shift+drag one of the corners or use shift+command+drag to scale from the center


  • use rulers to better align images with each other
  • command + R blends in the ruler toolbars
  • afterwards you can drag rulers by clicking on the tool bar and dragging them down or to the right


Additional Cropping of Individual Images


  • if you find yourself in the need to further crop an individual image, you can still do it in the following way
  • select the Rectangular Marquee Tool from the tools
  • select the area you want to cut out
  • then hit the delete button

Important: if you have the layer with the other image selected, the delete will not have any effect.
Make sure you have the right layer selected (see below).


  • once you are done, use the 'crop' took again to cut off the remaining canvas

    • Reducing File Size & Exporting to Illustrator


    • save the image as a .psd (photoshop file)
    • now before you export the final jpg, make sure you reduce the file size
    • many original photos have 5000pixels or more with the resulting jpg being 5MB
    • if we have 20 figures in the paper, this is a pdf of 100MB in the end -- not cool
    • to reduce image size, go to Image -> Image Size, the width does not need to be more than 1400px for looking good in a paper column



    • now you are ready to save the final jpg for use in Illustrator



    Import File into Illustrator


    • please use the template file example-figure.zip that we provided for you
    • in the folder, the Illustrator file is the pdf (it was exported as a pdf with Editing Capabilities)
    • open the pdf file in the folder in Illustrator
    • for your convenience, the file already has an image in it, but please replace it with you own
    • simply delete it and drag your own image onto the canvas, then rescale the image


    Please pay close attention to the following things:
    • the pair of rulers represents the width of one column in the paper, do not make your figure wider than the column, if you do, the a/b/c/ bubbles and all other text on the figure will be tiny because the figure will be downsized as you paste it in the paper
    • we always use the same a/b/c labels, you can copy them and rename if you need d/e/f
    • all text on figures is Arial 9pt
    • we will do more Illustrator in the next weeks, so if your figure looks something like this, you are fine


    Exporting the Figure for the Paper


    • go to File -> Export -> Export As
    • name the file and save as .png
    • select either 300 or 150 from the dropdown (300 will give you a larger file but higher res)



    • When you export, make sure there are no invisible floating figures/texts, otherwise your png will not be exported correctly
    • below, on the left side we forgot to delete the c label in the bottom right corner, thus the figure is weirdly large
    • deleting the (c) gives you the right side
    • whenever your image comes out larger than it's supposed to be, there is still something floating on the canvas (perhaps some visible line with no line color)



      • Inserting the Figure in the Paper


      • simply open the word file and drag the .png file that you exported in the right location
      • again note how on this page all the abc and text is the same size because we kept the layout of our figure within the columns



    Saving an Organizing


    • Each figure should be saved inside a folder by its number and name in this format: FigXX Title of figure.
    • Here is an example of how the Figures folder structure should be



    Each figure folder should have The image exported to PNG Here is an example
    • the original unmodified high resolution images used in the figure (.CR2 raw)
    • the Photoshop file of the composed figure that consists of multiple images (.jpg, or .psd)
    • the Adobe Illustrator file with figure and a/b/c labels (.pdf file saved in editable PDF format)
    • the exported file that will go into the paper (.png)
    • any extra files go into #more, any old files go into #old





    Re-Editing


    • If you need to edit the figure, open the PDF file in Illustrator and make changes.
    • Save the file as PDF.
    • Export the figure as PNG and replace the older PNG file in the folder.
    • the exported file that will go into the paper (.png)
    • Re-insert the figure in the paper.

    Re-Linking Images in Illustrator after Renaming/Moving Files


    • whenever you rename or move one of the jpgs that were used in the illustrator file, Illustrator will no longer be able to find that file and will throw a warning
    • to relink it, go to Window -> Links
    • in the popup, click the 'relink' icon and select the new location of the file
