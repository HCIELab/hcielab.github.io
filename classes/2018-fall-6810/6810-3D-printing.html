Install the Slicing Program (Cura)
  • Cura: https://ultimaker.com/en/products/ultimaker-cura-software
  • a slicing program converts a 3D model into layers for 3D printing and also generates the support material under steep overhangs to hold them up

Setup the Slicer Cura for the 3D Printer Prusa i3 Mk2
  • open Cura, click on the black bar in the top right corner that says Ultimaker
  • select 'Add Printers'
  • in the popup, click on 'others'
  • scroll down the list until you see the 'Prusa i3 Mk2'






  • the black bar should now say 'Prusa i3 Mk2'
  • scroll down in the settings and set the Printing Temperature to 210C
  • scroll down and check that the print speed says 60mm/s and travel speed (print head moving but not extruding) is set to 120mm/s
  • finally, scroll down and change the support placement to say 'touching the build plate' (support will only be generate where you have a chance to actually break it off and not inside in small holes)




Find a 3D Model
  • Thingiverse: https://www.thingiverse.com/
  • 3D models for 3D printing come as .stl file format

Slice your 3D Model
  • now load your 3D model into Cura (just drag the stl file onto the build plate), I used the Shroom.stl file
  • the red areas show you where support material will be generated eventually
  • switch from 'Solid View' to 'Layer View'
  • this will start the slicing process (takes a few minutes) and will show you the support material, the time it will take for printing etc.
  • you can use the 'move' and 'scale' functionality on the left side to position your model on the build plate (in case you want to print multiple things) and change the size of your model (in case you want to save time)




Export your sliced model for 3D Printing
  • go to File -> Save As
  • save as .gcode file
  • the 3D printers take SD cards, remove the SD card and add your gcode file, then plug the SD card back in




Print Your Model
  • the initial screen shows you some stats, i.e.
  • the state of the 3D printer at the moment (extruder temperature, build plate temperatur, extruder height)
  • and information about the currently running print (or last print if it was completed), e.g. minutes it took to complete and how much % is done




  • do not push the 'x' button, this is the on/off button for the 3D printer (not a backwards button)
  • pushing down on the handle (circular area) gets you to the main menu
  • rotate the handle to go through the menu options




  • to print using the already loaded material, scroll down to 'print from SD'
  • then select your .gcode file




Unload Filament
  • before unloading the currently filament, you have to pre-heat the extruder, otherwise you get a an error message
  • thus, first go to the main menu and select 'preheat', then enter ca. 200C
  • once the start screen shows that the temperature reached 200C, you go back to 'Unloading Filament' and now the error should be gone
  • as soon as you hear the click, grab the filament and pull it out
  • before putting it on the shelve, cut off the squished part of the front (this will create problems when you load it next time) to leave a clean end





Load New Filament
  • loading new filament works very similar
  • make sure the extruder is still preheated sufficiently (this should be the case if you just unloaded filament, but in case there was nothing loaded at the beginning you might have to do it now)
  • go to 'Load filament'
  • check that your filament has a clean end and then insert it
  • you should feel a slight pull from the extruder
  • if you look at the bottom of the extruder, you should see material coming out
  • if you change from, e.g. blue to yellow, there might still be a bit blue coming out before you see the yellow coming, confirm in the interface once you see your new color and you are all set




Remove your Print from the Build Plate
  • never scrape your print from the build plate when its done
  • just pull on it gently or bend the build plate in both directions, it should just come off
  • scratches on the build plate can make it hard for new prints to stick to the plate (i.e. the print will fail)
  • you can use glue stick and apply it thinly across the build plate to make prints stick better if you have trouble

GCode

  • gcode is the programming language for 3D printers (and other fabrication tools)
  • open the gcode file in a text editor of your choice (e.g. Sublime)
  • can you decode what the printer is doing in each line?
  • this might help: https://all3dp.com/g-code-tutorial-3d-printer-gcode-commands
  • or this: https://www.simplify3d.com/support/articles/3d-printing-gcode-tutorial
  • here's something that Stefanie did with custom gcode (first videom, called Wireprint, moves the print head up and down for faster printing, read more here if you are interested)
  • Gcode is also used for telling laser cutters what to do, the second video shows something that Stefanie did with custom gcode for laser cutters, called Destructive Games, read more here if you are interested
