In-Class Laser Cutting


Step 1: TA shows you how the laser cutter works (groups of 5, ca. 10 min)

 The TA will show you how the laser cutter works.
For this, we will split the groups in 5 students each (each TA has 2 groups).
If you are not in the first group, start with Step 2.

Step 2: You laser cut your own 2D drawing.

 Next, we will test if your drawing program has the right settings setup.
Open the drawing program you previously installed.
Create a simple 2D geometric shape (e.g. a rectangle, a circle) or reuse what you already had from the HW.
Once you are done, export your drawing as a .pdf file and email it to your TA.

Make sure that your drawing has the following settings otherwise the laser cutter will not be able to process it:

OpenDraw:
  • from the outline colors, select 'Red'
  • click infill and say 'None'



Adobe Illustrator:
  • set document mode to RGB (Menu: File -> Document Color Mode -> RGB)
  • when you create your shape, make sure there is only a line color but no infill
  • the outline needs to have pure red as color (R: 255, G: 0, B:0)
  • the stroke thickness should be 0.00001 (Window -> Stroke -> Weight), the moment you hit enter it will snap to 0 but internally it will still be 0.00001 so you should be all set



InkScape:
  • if you don't see your object, make sure opacity is 100%
  • set stroke width to 0.018mm ()
  • then disable stroke width auto-scaling: Edit -> Preferences -> Behavior -> Transforms -> Uncheck "Scale stroke width."

Step 3: fill out team partner survey

 Please fill out the team partner survey.

Step 4: Start working on HW, upload first version by end of class.

 Start working on your homework, you can find it here.
At the end of class make sure you upload a first version of your file to gradebook (pass/no pass).

Step 5: learn how to use a strip heater.

 If you are today in EDS or IDC, we will show you how to use the strip heaters there (you will need this later for pset1).
For the students in Stefanie's lab space and the CSAIL machine shop, we will show you how to use the strip heater next week in the 3D printing skills lab.
