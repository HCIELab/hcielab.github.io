Slicing for Dual Material Printing


To convert a 3D model for 3D printing, you have to load it into a software called a Slicer.
A slicer takes your 3D model and slices it horizontally into a number of layers.
Each layer is then converted into a print path (a long line filling the layer) that the print head follows while laying down material.

The slicer we are going to use is called Cura for Ultimaker and you installed it prior to this lab.
Note that most branded 3D printers have their own slicer (e.g. the MakerBot 3D printer uses the MakerBot slicer).
However, there are also slicers that can be used for many different 3D printers, e.g. a slicer called Slic3r supports this.

Open Cura and load both of the 3D models from Thingiverse into the view.
If they are misaligned, make sure to align them on top of each other by setting each of the 3D models x,y,z coordinates to 0,0,0.



Next, let's make sure we are going to use the correct 3D printer settings for slicing, i.e. there are different Ultimakers, e.g. Ultimaker 1,2,3, S5 etc and they all need different settings.
Click on the top left corner on the 3D printer name, then 'manage printers', in the opening window click on 'add' to add a new printer, and then open the 'add a non-networked printer' dialog.
In the dialog window, select the Ultimaker 3 Extended, which is the printer we are using for the class. Confirm the selection by clicking 'Add' and make sure that once the window closed, under 'local printers' the Ultimaker 3 Extended is now selected. Click 'close'.






Now that you have the 3D printer configured, we are ready to setup the materials for 3D printing, i.e. tell the slicer which extruder head will hold which material.
In our setup, we are using extruder 1 with regular white PLA filament, and extruder 2 with conductive PLA filament.

To do this, click on the top where it says (1) Generic PLA and select the white PLA from the list (see below).

Repeat the process with (2) and assign the conductive PLA (Michael: it is not in the list of materials, we may have to install the settings file for this first).




Now that you have assigned each extruder a material, you are ready to go to the next step: assign your model geometry to an extruder and thereby to a material. Click on one of the two meshes and in the left toolbar, choose either 1 or 2 depending on which material you want to print it with. The visualization should update accordingly, i.e. in the view you now see that the inner part is colored black since it will be printed from the black conductive material.



We are all set for slicing now!
Click the slice button on the right side.
Once slicing has finished, it should tell you how long the model will print and how much material it will require.
Note that if you go to print settings and you change the quality from e.g. 'fast' to 'fine' the print time will more than double.
However, at the same time, you will also get a higher quality print since 'fine' uses a thinner layer height than 'fast'.



Once you sliced, you can either save the file or preview it.
Let's look at preview for a moment.
The vertical slider allows you to step through each layer of your print and check if everything looks correct.



The horizontal 'play' bar then plays an animation of how the print head will print that particular layer, pretty neat!



Once you are sure everything looks good, you can save the file.
Note that the file extension is .gcode, which is what the 3D printer takes as input.
