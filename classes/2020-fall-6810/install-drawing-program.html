2D Drawing Programs

You will need a drawing program both for making drawings that can be used for conductive injket printing of circuits as well as for making drawings for laser cutting.

Please install one of the 2D drawing programs from below.

Recommended:
  • Adobe Illustrator (but only 30 day test version)
  • OpenDraw
  • Inkscape


If your drawing program doesn't open because of your 'security preferences' (mostly on Mac), you can go to your systems settings (Apple -> System Preferences -> Security and Privacy, then at the bottom of the window say 'open anyway').

Create a drawing with some geometric shapes (random example below) using the program of your choice and upload it to gradebook by the deadline (just pick some shapes, this is only to make sure you are ready for the lab.
