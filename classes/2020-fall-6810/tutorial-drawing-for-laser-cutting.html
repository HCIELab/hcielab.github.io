Drawing for Laser Cutting

(1) Use the Correct Line Color and Thickness

Most laser cutters require that you prepare your drawing for laser cutting by using specific line colors and thicknesses.
The line colors will be used to encode different power/speed settings for the laser and the line thickness will tell the laser which lines you want to cut along and which lines should be engraved as an image ('raster mode').

In vector mode a line is treated as a path along which the laser cuts. This results in a smooth laser motion along the outline of the mug.
In raster mode a line is treated as an area and each pixel in that area will be engraved/rastered, the laser does not follow the outlines of the shape and thus the design is not cut.

Thus, make sure that your drawing has the following settings otherwise the laser cutter will not be able to process it:

OpenDraw:
  • from the outline colors, select 'Red'
  • click infill and say 'None'



Adobe Illustrator:
  • set document mode to RGB (Menu: File -> Document Color Mode -> RGB)
  • when you create your shape, make sure there is only a line color but no infill
  • the outline needs to have pure red as color (R: 255, G: 0, B:0)
  • the stroke thickness should be 0.00001 (Window -> Stroke -> Weight), the moment you hit enter it will snap to 0 but internally it will still be 0.00001 so you should be all set



InkScape:
  • if you don't see your object, make sure opacity is 100%
  • set stroke width to 0.018mm ()
  • then disable stroke width auto-scaling: Edit -> Preferences -> Behavior -> Transforms -> Uncheck "Scale stroke width."


(2) Export your Drawing as .svg or .pdf

The laser cutter can take as input either an .svg file or a .pdf file.
So make sure that you export your file in one of those formats before you put it in your google drive.

(3) Want to Cut and not Engrave Text?

Your business card may have text that you want to cut out and not engrave.
However, if you use the text tool in your drawing program, your drawing program will likely treat the letters not as a 'path' (what you want) but rather as a filled area (what you don't want), i.e. those will not be cut out but engraved. So how can we fix this? We need to convert our text to be a 'path'.

Adobe Illustrator
1. Create your text using the Type Tool.


2. Select your text using the Selection Tool.


3. Go to "Type" - "Create Oulines" to turn your text into outlines.


4. For laser cutting, set the infill to none, outline to red (#ff0000), and stroke to 0.00001 pt (it will show up as 0).


OpenDraw
1. Create your text using the Text Tool.


2. Go to "Modify" - "Convert" - "To Contour" to turn your text into outlines.


3. For laser cutting, set the infill to none, outline to red, and line style to continuous.


Inkscape
1. Create your text using the Text Tool.


2. Select your text using the Selection Tool.


3. Go to "Path" - "Object to Path" to turn your text into outlines.


4. For laser cutting, set the fill to none, stroke color to red (255), stroke width to 0.018mm. Then disable stroke width auto-scaling: Edit -> Preferences -> Behavior -> Transforms -> Uncheck "Scale stroke width."
